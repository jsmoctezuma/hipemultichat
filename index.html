Qué hace

Pensado para tener tus chats principales en una sola fuente de OBS, con control visual desde un panel local.

💬

Chat unificado

Muestra mensajes de varias plataformas en un solo overlay para tus escenas.

🎛️

Panel visual

Genera URL para OBS, cambia colores, fuente, tamaño, opacidad y filtros sin tocar código.

🎁

Eventos y regalos

Soporta eventos como subs, bits, raids, gifts, joyas de YouTube y regalos de TikTok.

🧼

Filtros por plataforma

Activa o apaga chat, regalos, follows, likes/taps y otros eventos según lo necesites.

🧩

Dos formatos

Horizontal para ticker inferior compacto y vertical para chat completo en columna.

✨

Emotes y badges

Renderiza emotes, badges y avatares cuando la plataforma entrega esos datos.

🛡️

Moderación pasiva

Respeta eventos de moderación disponibles por plataforma sin mezclar chats entre servicios.

⚡

Local para OBS

Diseñado para correr como archivos locales con Browser Source y conexiones locales.

Plataformas

Chats soportados

Cada plataforma se puede prender o apagar desde el Admin. Algunas funciones dependen de lo que cada servicio entrega realmente.

Twitch

Chat + eventos

Mensajes, badges, emotes, bits, subs, raids, puntos del canal, PowerUps y moderación disponible.

YouTube

Chat + monetización

Chat, Super Chat, Super Sticker, membresías, regalos y JewelsGifted en formato horizontal o compacto.

Kick

Chat + eventos

Chat, follows, subs, regalos y eventos disponibles por Streamer.bot. Delete/clear chat depende de lo que Kick proporcione.

TikTok

TikFinity

Chat y regalos por WebSocket local de TikFinity. Likes/taps se controlan para evitar spam.

Uso rápido

Cómo usarlo

Este index no incluye importaciones de Streamer.bot. El punto principal es abrir el panel, generar la URL y probar en OBS.

Abre el panel

Entra a admin.html y elige horizontal o vertical según la escena donde lo quieras usar.

Configura filtros

Prende o apaga Twitch, YouTube, Kick y TikTok, además de los eventos que quieres mostrar.

Copia la URL

Usa el botón del Admin para copiar la URL lista para OBS con tus parámetros actuales.

Pégalo en OBS

Agrega una Browser Source local y prueba primero con connect=0, luego con conexión real.

Formatos

Horizontal vs vertical

No son copias exactas. Cada modo tiene intención distinta para no romper el diseño en OBS.

TikTokRose x5Twitchmensaje compacto

Horizontal

Ticker inferior compacto. Resume eventos, mantiene una sola línea y evita tarjetas grandes o imágenes que rompan la barra.

Vertical

Chat completo en stack/columna. Puede mostrar mensajes, tarjetas de eventos, gifts con más detalle, previews y elementos más visuales.

Ayuda

Notas importantes

Puntos útiles antes de probarlo en una escena real de OBS.

¿Necesita Streamer.bot?

Para Twitch, YouTube y Kick se usa la conexión WebSocket de Streamer.bot. Este index no trae bloque de importación ni instalación de acciones; solo documenta el uso del proyecto.

¿Cómo funciona TikTok?

TikTok entra por TikFinity usando su WebSocket local. TikFinity debe estar abierto para que el overlay reciba chat y eventos.

¿Por qué horizontal no muestra todo igual que vertical?

Porque horizontal es un ticker compacto. Lo importante es que no rompa la barra inferior y que la información se vea rápida y limpia.

¿Qué pruebo primero?

Abre horizontal.html?connect=0 y vertical.html?connect=0 para revisar diseño. Luego activa conexiones reales desde el Admin y pruébalo en OBS.

Listo para OBS

Abre el Admin, ajusta el overlay y copia la URL generada para tu escena. Prueba horizontal para barra inferior compacta o vertical para chat completo.
